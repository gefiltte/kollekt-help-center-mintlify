# Add and organize link groups

<Warning>
**This article is a stub.** Full walk-through coming soon.
</Warning>

Link groups are the sections on your Artist page that hold one or more links. Each group has a title, an optional description, and one or more links. Links have a title, a URL, and an optional image thumbnail.

## What you can do

- Create a new link group
- Add links to a group (title, URL, optional image)
- Reorder links within a group
- Reorder groups on the page
- Delete a group or a single link

## Tips

- Use clear group names fans recognize: **Merch**, **New music**, **Tour**, **Socials**.
- Add an image to each link when you can — they convert better than plain text rows.
- Keep 2–4 groups. More than that and the page starts to feel like a list.
